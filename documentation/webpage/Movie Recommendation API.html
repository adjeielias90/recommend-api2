Get Recomendations(id,title)

http://0.0.0.0:5000/api/filter/collaborative

get recomendation with movie title and userID Collaborative filtering

BODYraw
{
  "id": "1", 
  "title": "Star Trek Into Darkness"
}


Example Request
Get Recomendations(id,title)
curl --location --request POST 'http://0.0.0.0:5000/api/filter/collaborative' \
--data-raw '{
  "id": "1", 
  "title": "Star Trek Into Darkness"
}'

Get Top Movies

http://0.0.0.0:5000/api/top_movies

GET request that returns the top 10 json list of movies in our dataset.

Get Top Movies By Genre(Romance example)

http://0.0.0.0:5000/api/top_movies/romance

GET request that returns the top 10 json list of movies in particular genre. If not genre is supplied, it defaults to returning the top 10 movies of our dataset.



Example Request
Get Top Movies By Genre(Romance example)
curl --location --request GET 'http://0.0.0.0:5000/api/top_movies/romance'
