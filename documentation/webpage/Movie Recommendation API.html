Get Top Movies By Genre(Romance example)

http://0.0.0.0:5000/api/top_movies/romance

GET request that returns the top 10 json list of movies in particular genre. If not genre is supplied, it defaults to returning the top 10 movies of our dataset.



Example Request
Get Top Movies By Genre(Romance example)
curl --location --request GET 'http://0.0.0.0:5000/api/top_movies/romance'
